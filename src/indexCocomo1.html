Choose type of your project:

Organic Mode: Relatively small. simple software projects in which small teams with good application experience work to a set of less than rigid requirements.
Semi-detached Mode: An intermediate. (in size and complexity). software project in which teams with mixed experience levels must meet a mix of rigid and less than rigid requirements.
Embedded Mode: A software project that must be developed within a set of tight hardware. software and operation constraints.
